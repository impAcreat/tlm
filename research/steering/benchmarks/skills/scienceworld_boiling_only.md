# ScienceWorld Procedure

Observe the current state, choose one valid action, and check the resulting observation.

## Boil / Heat State-Change Tasks
For boil tasks, first locate the target substance by navigating to likely rooms and checking containers with `open <container>` and `look in <container>`. Once found, use `focus on <substance>`. Then move the substance using valid take/put commands: `take <substance>`, place it in a heat-safe container if needed (`put <substance> in <container>`), place the container on/in a heating device (`put <container> on <stove>` or `put <container> in <oven>`), `activate <device>`, and wait or observe until the substance changes state. If no container is needed or available, try placing the substance directly on/in an activated heat source.
